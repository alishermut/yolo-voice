# YOLO Voice v1.0.3: Support Bundles and Deeper Output Diagnostics

## What's new

- Adds an in-app support diagnostics flow so users can enable extra logging and export a local support bundle without sharing audio recordings or API keys.
- Moves the support tools onto the `About` screen with controls to enable diagnostics, export debug logs, and clear previous diagnostics.
- Expands runtime tracing across the dictation pipeline so support bundles now show recording, capture, transcription, finalize, and insert stages step by step.
- Adds detailed output insertion diagnostics for Windows, including target process metadata, window class, elevation state, foreground-window transitions, and `SendInput` results.

## Why this release matters

- This release is aimed at real-world support. When dictation works but text insertion fails on another machine, the exported bundle should now tell us whether the issue looks like focus loss, privilege mismatch, or a blocked synthetic input path.
- It also gives users a safer support workflow because diagnostics stay local until they choose to export and share the bundle manually.
