# YOLO Voice v0.9.6: Distil-Whisper Offline Upgrade

## What's new

- Replaces the old secondary offline model with `Distil-Whisper`, an English-focused whole-clip option with stronger raw dictation quality than Parakeet.
- Keeps `Parakeet` as the default offline model and adds a `Fast segmented mode` toggle so you can choose between maximum responsiveness and cleaner one-shot transcription.
- Adds automatic background preparation for Distil-Whisper, so selecting it starts loading the model without an extra manual step.
- Adds CPU/GPU switching for Distil-Whisper in Settings, matching the existing Parakeet acceleration controls.
- Improves the offline model details panel so it reflects the real Distil-Whisper support story: English-first, GPU recommended, CPU available but slower.
- Makes recording start feedback more trustworthy by playing the start sound only after recording is actually ready.
- Adds a `Clear all history` action in transcript history for quick manual cleanup.

## Why this release matters

- Offline English dictation now has a clearer two-model setup:
  - `Parakeet` for fast, broad-device dictation
  - `Distil-Whisper` for higher-quality English whole-clip transcription
- The speed-versus-quality tradeoff is now explicit for Parakeet instead of being hidden behind one fixed pipeline.
- Distil-Whisper feels more productized because it prepares automatically and supports the same CPU/GPU switching flow as the default model.
