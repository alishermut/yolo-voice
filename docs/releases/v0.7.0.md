## What's new

- Refactor dictionaries into a scoped personal-dictionary plus active-industry-pack model, with legacy merged dictionaries backed up and reset on migration.
- Improve deterministic cleanup and segment recombination so VAD chunk boundaries no longer become sentence breaks by default.
- Add an opt-in local SQLite transcript diagnostics store for inspecting raw segments, joined text, cleaned text, final text, and insertion outcomes.
- Make Personal Terms more useful by generating safe automatic aliases for common shape-based variants such as `type script -> TypeScript` and `next js -> Next.js`.

## Why this release matters

- Industry packs now behave predictably without leaking old pack rules into unrelated domains.
- Offline dictation reads more naturally because cleanup is less destructive and segment joining is much smarter.
- Real dictation sessions can now be debugged locally with structured transcript logs instead of relying only on synthetic evaluation.
- Technical product and service names are easier to preserve without requiring every common spacing variant to be entered manually.
