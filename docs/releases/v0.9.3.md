# YOLO Voice v0.9.3: Auto-Update & Data Persistence

## What's new

### In-app auto-update
- The app now checks for updates automatically 10 seconds after launch
- A banner appears at the top of Settings when an update is ready, with "Install & Restart" to apply it in one click
- The About section has a full updater panel: check manually, see download progress, install when ready
- A green pulse dot on the About sidebar item signals when an update is available
- Updater UI translated across all 15 supported languages

### User data preserved on reinstall
- Settings, dictation styles, vocabulary, and API keys survive app reinstalls
- On Windows uninstall, a prompt now asks whether to delete user data or keep it for future use
- Choosing "No" keeps everything in place so a reinstall picks up right where you left off

## Cleanup

- Removed unused legacy files: old About page, standalone updater hook, EqualizerBars, LLMSettings
- Removed stderr log redirect and engine error file logging (diagnostics section removed)
- Updated author metadata
