# YOLO Voice v1.0.6: Distil Sidecar Pinning Fix

## What's new

- Fixes the Distil-Whisper production startup error caused by Python sidecar dependency drift inside the packaged app.
- Pins the bundled Distil base runtime to the exact tested versions:
  `torch 2.6.0`, `transformers 5.5.3`, `huggingface-hub 1.10.1`, and `soundfile 0.13.1`.
- Tightens `prepare-sidecar.ps1` so cached sidecar environments are rebuilt unless those exact versions are present, preventing release runners from silently reusing a mismatched bundle.
- Keeps the `v1.0.5` on-demand CUDA runtime flow, so Distil GPU still installs its larger CUDA runtime only when the user explicitly switches to GPU.

## Why this release matters

- `v1.0.5` fixed the packaging shape for Distil CUDA, but the bundled CPU base runtime was still allowed to float on Python dependencies.
- In production, that can surface as Distil failing to prepare with import errors inside the shipped sidecar environment.
- `v1.0.6` makes the bundled Distil runtime deterministic and matches the exact package set that was verified locally.

## Verification
