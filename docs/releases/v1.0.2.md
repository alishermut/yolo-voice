# YOLO Voice v1.0.2: Publishable Distil GPU Runtime

## What's new

- Fixes the Windows release pipeline by shrinking the bundled Distil-Whisper runtime back to a CPU-only base package that fits inside the NSIS installer again.
- Adds on-demand Distil-Whisper GPU provisioning on Windows, so the app can prepare a cached CUDA runtime in app data the first time GPU is requested instead of bundling several gigabytes of CUDA libraries into every installer.
- Surfaces progress messaging during Distil-Whisper preparation so GPU setup and runtime switching are easier to understand in the UI.
- Tightens Distil-Whisper GPU behavior so an explicit GPU request now fails clearly if CUDA could not be provisioned, instead of silently dropping back to CPU.

## Why this release matters

- `v1.0.1` fixed the runtime logic but could not be published because the installer became too large for NSIS after bundling the full CUDA PyTorch stack.
- `v1.0.2` keeps the same end-user goal, real GPU acceleration for Distil-Whisper, while making the release artifact small enough to publish and update reliably.
