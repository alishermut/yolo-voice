# YOLO Voice v1.1.1: Parakeet English Stability Fix

## What's new

- Fixes a Parakeet offline transcription issue where English speech could be decoded as Russian/Cyrillic text.
- Vendors a small patched `parakeet-rs` build so YOLO Voice can seed the Parakeet TDT decoder with English and suppress later language-switch control tokens.

## Why this release matters

- English dictation should now stay in English more consistently on the Parakeet path.
- Multilingual Parakeet auto-detection is intentionally constrained by this fix; use Distil-Whisper or cloud transcription for multilingual dictation until an app-level language mode selector is added.
