# YOLO Voice v1.0.0: Richer Sounds, Reliable History, and More Stable Hotkeys

## What's new

- Expands the bundled UI sound library with a much larger CC0-safe set from OpenGameArt alongside the earlier `soundcn` additions, giving start and stop feedback many more polished options.
- Fixes transcription History so new entries are saved again even when the hidden diagnostics flag is off, and command-mode results now appear in History too.
- Surfaces History load failures in the UI instead of failing silently.
- Stabilizes dictation hotkeys by replacing the old double-tap wait heuristic with a real delayed-stop timer, so short press-and-release no longer depends on unrelated future input to stop recording.
- Tightens hotkey state reconciliation and style-key filtering so recordings are less likely to get stuck and random repeated keys no longer spam style-switch paths.

## Why this release matters

- The app now feels more polished in everyday use: sounds are nicer, History is trustworthy again, and recording controls behave much more predictably.
- This is a good `1.0.0` milestone because the core loop of recording, transcribing, reviewing, and controlling the app from the tray is meaningfully more stable than the earlier 0.9.x builds.
