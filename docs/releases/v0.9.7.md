# YOLO Voice v0.9.7: Updater Reliability and Distil-Whisper Packaging Fixes

## What's new

- Fixes Windows auto-update metadata so `latest.json` now points to the actual published installer asset instead of a guessed filename.
- Adds release-time verification for updater downloads, which catches broken release URLs before a tag ships.
- Bundles `distil_whisper.py` into the Tauri app resources and teaches the runtime to find it from packaged installs.
- Improves update installation error handling in the app so failed installs surface a real error state instead of silently stalling.
- Adds release validation for bundled Distil-Whisper resources to prevent shipping another broken installer.

## Why this release matters

- In-app updates should work again for Windows users because the updater metadata now matches the real GitHub release assets.
- Packaged builds can launch Distil-Whisper correctly without depending on repo-root paths that do not exist after installation.
- The release pipeline now validates both updater artifacts and Distil-Whisper resources, which makes future releases much safer.
