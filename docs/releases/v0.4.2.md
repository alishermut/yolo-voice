## What's fixed

- Fix offline transcription in installed builds by resolving bundled sidecar files from the packaged `sidecar` resource directory.
- Fix bundled industry pack lookup in installed builds.
- Fix updater publishing so every release uploads `latest.json`, the signed `.nsis.zip`, and the matching `.sig` file.

## Why this release matters

- Installed production builds could show `Transcription engine: stopped` even though dev builds worked.
- Manual update checks could fail because the release was missing the updater metadata and signed update bundle.

## Verification

- Verified the packaged installer now includes the expected sidecar resources and that the app resolves them from the correct production path.
- Verified the release workflow now searches the NSIS bundle directory recursively, logs the discovered artifacts, and uploads the updater assets with explicit quoted paths.
