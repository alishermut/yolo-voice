# YOLO Voice v0.9.9: Better UI Sounds and a Smarter Tray Menu

## What's new

- Replaces the old built-in sound set with a curated batch of cleaner CC0 UI sounds imported from the `soundcn` catalog's Kenney assets.
- Adds several new bundled sound options for recording start and stop feedback, with MP3 playback support on Windows through `rodio`.
- Expands the tray menu to include `Show app`, `Transcriptions`, `Settings`, `History`, and `Quit`.
- Makes tray navigation smarter by opening the main window directly to the Transcription or History section when those menu items are selected.
- Fixes the tray right-click behavior so the context menu stays open instead of immediately disappearing.

## Why this release matters

- Recording feedback sounds more polished and gives users more choices without introducing licensing risk for the open-source project.
- The tray menu is now much more useful for quick access, and the right-click menu behaves the way users expect.
