# YOLO Voice v1.0.1: Distil-Whisper GPU Fixes and Sidecar Packaging Repair

## What's new

- Fixes the Distil-Whisper packaging path so release builds bundle the Python sidecar runtime instead of falling back to stale cached environments.
- Repairs Distil-Whisper status handling in the UI so failed prepares stay in `error` instead of flickering through `downloaded` and re-triggering auto-prepare loops.
- Switches the bundled Distil runtime to a CUDA-enabled PyTorch build on Windows, so selecting GPU now loads Distil-Whisper on `cuda:0` when CUDA is available.
- Fixes the legacy faster-whisper CUDA probe to use the current `ctranslate2` API, which stops false CPU fallbacks on machines with a working NVIDIA runtime.
- Strengthens the sidecar build and release verification steps so broken or CPU-only bundled runtimes are caught before a release is published.

## Why this release matters

- The production app now ships the runtime Distil-Whisper actually needs, instead of depending on whatever stale sidecar cache happens to exist on the machine.
- GPU acceleration is now wired end to end for both offline Whisper paths on supported Windows NVIDIA systems, which removes the biggest performance regression reported after `v1.0.0`.
