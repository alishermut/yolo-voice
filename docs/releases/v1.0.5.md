# YOLO Voice v1.0.5: Distil-Whisper CUDA Hotfix

## What's new

- Restores real Distil-Whisper CUDA support on Windows without bloating the installer. The packaged app now ships a smaller bundled Python runtime, then prepares a cached CUDA runtime on first `Switch to GPU`.
- Keeps the earlier production fixes from `v1.0.4`, including the updater `_up_/sidecar` runtime lookup fix and the stable Distil error state instead of the old prepare-loop flicker.
- Adds progress messaging for the one-time Distil GPU setup, so users can see when the app is copying the bundled runtime, downloading CUDA torch, and verifying the GPU runtime.
- Returns Windows packaging to the normal NSIS path now that CUDA is no longer bundled directly inside the installer artifact.

## Why this release matters

- Previous production builds either stayed on CPU or became too large and unstable to package when CUDA was bundled directly.
- This release moves the large CUDA payload out of the installer and into an on-demand runtime cache, which keeps releases shippable while still letting Distil run on compatible NVIDIA GPUs.
