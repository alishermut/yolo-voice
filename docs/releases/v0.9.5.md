# YOLO Voice v0.9.5: Offline Dictation Refinement

## What's new

- Adds language-family aware offline cleanup so Cyrillic/Russian dictation avoids English-only text shaping.
- Keeps the improved offline preview assembly and diagnostics telemetry for mixed-script and utterance metadata.
- Rolls back the expensive second-pass finalize path so stop-to-insert latency stays snappy in everyday dictation.

## Why this release matters

- Mixed English/Russian dictation is safer because cleanup rules now respect script family instead of blindly applying English heuristics.
- The app feels more responsive again after removing the extra full-utterance re-decode that was adding noticeable delay.
- Release metadata, updater tagging, and app versions are now aligned on `0.9.5`.
