# YOLO Voice v0.9.8: Quieter Distil-Whisper Downloads and a Cleaner About Page

## What's new

- Hides the Windows terminal window that previously appeared while Distil-Whisper was downloading or preparing in the background.
- Silences Hugging Face progress-bar output during Distil-Whisper downloads and removes the deprecated download argument warning.
- Simplifies the About page by keeping the updater section and replacing the stack and shortcut summaries with the latest bundled release notes for the current app version.

## Why this release matters

- Downloading Distil-Whisper now feels like part of the app instead of launching a separate terminal window.
- The About page is more useful because it shows what changed in the installed version rather than repeating internal implementation details.

## Verification

- Passed `npm run build`.
- Passed `cargo test --quiet`.
- Passed `python -m py_compile sidecar\\distil_whisper.py`.
